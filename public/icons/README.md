# App Icons

Place the following PNG files here before shipping:

- `icon-192.png` — 192×192 px, used for Android home screen and PWA install
- `icon-512.png` — 512×512 px, used for splash screens and app stores

Both icons should have a transparent or solid background matching `#0A0E1A`.
The foreground should be the ParkSmart P logo on an indigo (#6366F1) rounded-rect.
